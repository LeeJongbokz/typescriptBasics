


- 타입스크립트 기본 타입에는 12가지가 있음  


```
// JS 문자열 선언
// var str = 'hello';

// TS 문자열 선언
let str: string = 'hello';
```
-> 이것이 타입스크립트에서 변수를 정의하고 선언하는 방식임  


```
// JS 문자열 선언
// var str = 'hello';

// TS 문자열 선언
let str: string = 'hello';
let num: number = 10;
let arr: Array<number> = [1,2,3]; 
let heroes: Array<string> = ['Capt', 'Thor', 'Hulk'];
let items: number[] = [1,2,3];
```
